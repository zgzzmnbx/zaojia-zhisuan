## 可借鉴点

- 左侧不是单层菜单，而是“极窄图标栏 + 一级模块栏 + 二级功能栏”的分层结构。
- 一级模块用业务分组组织，例如 Analyze / Leads / Manage；大尾巴主题可对应为填价、预览、预警、工作量、报告、知识库、设置。
- 当前选中项使用浅灰底和轻边框，不靠大面积高饱和颜色。
- 菜单文字短，图标轻，整体密度适合桌面办公系统。
- 主内容区与左侧菜单之间有明确分割线，减少视觉粘连。

## 不宜照搬点

- Navattic 是产品演示平台，页面存在较强营销和 demo 管理属性；造价智算不能照搬其业务文案和卡片模式。
- 三层左栏如果完整照搬，会挤压中间 Excel 表格宽度；造价智算需要控制左侧总宽度。
- 右侧在造价智算中应优先给“问问智算”AI 随行助手，而不是继续扩展更多左侧菜单层。

## 对大尾巴主题的建议

- 桌面宽屏可采用“三段式左侧导航”：
  - 极窄图标栏：产品、搜索、模块总入口、设置、用户。
  - 一级模块栏：填价工作台、结果预览、经验池预警、工作量抓取、Word 报告、知识库问答。
  - 二级功能栏：仅在当前模块需要时出现，例如“表2 / 表3 / 表4”“字段设置 / 隐藏空行 / 导出设置”。
- 普通笔记本宽度下应压缩为“两段式左侧导航”：
  - 极窄图标栏 + 可展开模块抽屉。
  - 中间主体和右侧 AI 优先保留。
- 左侧和右侧都必须可折叠，折叠后中间表格区域获得最大宽度。
- 菜单激活态建议采用浅蓝绿底、深蓝绿文字、1px 轻边框，避免高饱和大色块。
