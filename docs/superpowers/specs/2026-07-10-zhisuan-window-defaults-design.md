# 智算窗口集中默认配置设计

## 目标

将问问智算当前全部窗口设置统一纳入 `config/project-default-settings.json`，与表格预览、列映射和工作量抓取保持同一维护方式。默认横向宽度为 `400px`。

## 配置边界

新增 `zhisuanWindow` 分组，包含聊天区高度、窗口横向宽度、纵向高度跟随、快捷指令（内置开关、自定义指令、自动隐藏）、显示项、欢迎语和外观风格。

配置文件是项目默认的唯一来源。设置页中的调整仅作用于当前前端会话；重新打开页面或选择“恢复项目默认”时，重新读取该配置。智算窗口不再读写相关浏览器 localStorage。

## 数据流与兜底

后端按既有 `/api/project-default-settings` 接口下发已清洗的 `zhisuanWindow`。配置缺失或格式错误时，前端和后端分别使用内置兜底，其中窗口宽度固定为 `400px`。

## 验收

- `config/project-default-settings.json` 可直接维护全部智算窗口默认值。
- 清除或保留既有浏览器缓存均不会影响项目配置的加载结果。
- 设置页可临时调整，也可恢复项目默认；默认宽度显示为 `400px`。
- `npm run frontend:build` 通过。
