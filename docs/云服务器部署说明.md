## 运行方式

云端采用单端口网页版部署：

```text
FastAPI 后端：0.0.0.0:1285
React 前端：frontend/dist，由 FastAPI 静态托管
```

启动命令由 systemd 管理：

```bash
/opt/zaojiazhisuan/venv/bin/python -m uvicorn app.main:app --host 0.0.0.0 --port 1285 --app-dir backend
```

环境变量：

```text
PYTHONUTF8=1
GUANKAN_FRONTEND_DIR=/opt/zaojiazhisuan/current/frontend/dist
```

大模型 Key 通过 `/opt/zaojiazhisuan/current/.env.local` 读取；不要把 Key 写入源码或文档。

## 飞书第二层机器人

当前服务器已启用唯一的飞书企业自建应用长连接实例：

```bash
systemctl status zaojiazhisuan-feishu-bot.service --no-pager
systemctl restart zaojiazhisuan-feishu-bot.service
curl http://127.0.0.1:1285/api/collaboration/feishu-app-bot/status
```

机器人程序位于当前发布目录的 `backend/feishu_bot_runner.py`，依赖 `backend/requirements-runtime.txt` 中的 `lark-oapi`。企业应用凭证与第一层 Webhook 配置统一保存在服务器当前发布目录的 `Codex-Temp/runtime/feishu-robot-settings.json`，权限为 `600`，不进入 GitHub、发布压缩包或前端。

同一飞书应用不要让本机和服务器同时启用长连接。迁移到服务器后，本机第二层机器人应保持关闭，服务器只保留一个运行实例。

云端端口口径：服务器 FastAPI 监听 `1285`，因此服务器运行时配置中的 `feishuAppBot.apiBaseUrl` 必须为 `http://127.0.0.1:1285`；本机开发默认值仍为 `http://127.0.0.1:8000`，不要把云端端口改回本机默认值。端口不一致时，飞书文件任务和知识库问答均会出现 `[Errno 111] Connection refused`。

```bash
tail -n 100 /opt/zaojiazhisuan/logs/service-error.log
```

本机健康检查：

```bash
curl http://127.0.0.1:1285/api/health
```

## 更新部署口径

1. 本地执行 `npm run frontend:build`。
2. 生成云端部署包，包含：
   - `backend/app/`
   - `backend/requirements-runtime.txt`
   - `frontend/dist/`
   - 正式二维知识库
   - 报告模板
   - 经验池模板、经验池数据和偏好配置
   - `config/project-default-settings.json`：必须随云端包发布，统一承载表格预览、列映射、工作量抓取和问问智算窗口默认设置；可用 `_说明` 字段人工维护，不能使用 `//` 或 `/* */` 注释
  - 匹配规则说明、术语归并表和知识问答所需资料
   - `.env.local.example`
   - 如需云端直接启用问问智算，可随包带入 `.env.local`
3. 上传到 `/tmp/`。
4. 解压到 `/opt/zaojiazhisuan/releases/新版本目录`。
5. 将 `/opt/zaojiazhisuan/current` 软链接指向新版本目录。
6. 运行：

```bash
/opt/zaojiazhisuan/venv/bin/pip install -r /opt/zaojiazhisuan/current/backend/requirements-runtime.txt
systemctl restart zaojiazhisuan.service
```

### 中文路径打包注意事项

云端发布包包含中文目录和文件名，必须使用 UTF-8 保真的 `tar.gz` 上传并在 Linux 服务器上用 `tar -xzf` 解压。不要使用 Windows `Compress-Archive` 生成的 ZIP 直接在服务器上解压，否则可能把中文路径解码为乱码，造成知识库、报告模板和规则资料“文件不存在”。

## 注意事项

- 当前云端包不包含样例 Excel，样例由使用者自行准备。
- `config/project-default-settings.json` 是云端运行必需配置，必须与 `backend/app/` 位于同一发布根目录；缺失或 JSON 格式错误时程序会回退到内置默认值，可能与项目维护口径不一致。
- 云端部署不改变基价 / 单价匹配、调整系数匹配、经验池预警、工作量抓取和 Word 报告规则。
- 如果公网无法访问但服务器本机 `curl 127.0.0.1:1285` 正常，优先检查云服务器安全组是否放行 `1285/tcp`。
