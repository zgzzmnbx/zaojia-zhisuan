# 造价智算 Windows 评委运行版说明

## 第一阶段方案

评委运行版采用“FastAPI 后端托管静态前端 + Windows 启动脚本”的绿色目录方案：

- 评委双击 `启动造价智算-评委版.bat`。
- 启动器自动读取 `.env.local`，优先使用随包 `runtime/python/python.exe` 和 `runtime/python-libs` 本地依赖，启动 `127.0.0.1:8000`。
- 如需启用大模型，评委可先双击 `配置大模型Key-评委版.bat` 自动生成并打开 `.env.local`；只修改 `.env.local.example` 不会生效。
- 后端同时提供 API 和前端页面，浏览器访问 `http://127.0.0.1:8000/`。
- 前端构建产物放在绿色目录 `web/`，不需要评委安装 Node。
- 未配置 `DEEPSEEK_API_KEY` 时，智算问答 / 风险报告会提示大模型不可用，不影响 Excel 匹配、Excel 下载、Word 报告、经验池预警和工作量抓取的本地规则流程。

## 进入评委版的内容

- `backend/app/`：FastAPI 后端和本地规则执行代码。
- `backend/requirements-runtime.txt`：评委版运行依赖。
- `web/`：由 `frontend/dist` 生成的静态前端。
- `03-知识库-二维数据库制作/【数据库】【导入】.xlsx`：基价和第二层经验提示知识库。
- `03-知识库-二维数据库制作/01-报告模板-招标控制价报告模板/`：Word 报告模板。
- `03-知识库-二维数据库制作/【项目例子】*.xlsx` 与 `【委托方例子】*.xlsx`：评委现场快速验证主流程和工作量抓取的示例文件。
- `03-【匹配规则】-勘察测绘知识库-匹配规则提炼/` 中的核心规则文档、标准资料 Markdown / 批注表、技术工作费规则说明资料。
- `05-经验池-预警数据/` 中的经验池模板、当前经验池和字段偏好设置。
- `README.md`、`AGENTS.md`、`CHANGELOG.md`、`.env.local.example`、评委版启动 / 停止 / 状态 / 大模型配置脚本。
- `runtime/python/`：随包内置的官方 Windows embeddable Python，评委电脑不需要单独安装 Python。
- `runtime/python-libs/`：随包预装的本地 Python 依赖目录。
- 可选：`runtime/wheels/` 离线 Python 依赖包，用于本地依赖目录丢失时重建环境。

## 不进入评委版的内容

- `frontend/src/`、`frontend/node_modules/`、Vite 开发服务文件。
- `backend/tests/`、`.pytest_cache/`、`__pycache__/`。
- `04-输出版本存档/` 历史版本包。
- 参赛 PPT、演示视频、汇报素材、真实项目过程资料、大量调试输出。
- 本机 `.env.local` 和任何真实 API Key。

## 大模型配置排障

绿色版默认不携带真实 API Key。目标电脑如果需要使用“问问智算”、知识库问答或风险报告，请在解压后的评委版根目录操作：

1. 双击 `配置大模型Key-评委版.bat`。
2. 在自动打开的 `.env.local` 中填写 `DEEPSEEK_API_KEY=你的Key`，保存后关闭记事本。
3. 如果程序已经启动，先双击 `停止造价智算-评委版.bat`，再双击 `启动造价智算-评委版.bat`。
4. 双击 `检查造价智算状态-评委版.bat`，确认显示 `.env.local: 已找到` 且 `DEEPSEEK_API_KEY: 已读取`。

常见误区：

- 只修改 `.env.local.example` 不会生效；后端读取的是 `.env.local` 或进程环境变量。
- 在 Windows 系统环境变量里新增 Key 后，已经启动的后端进程不会自动继承，必须停止后重启。
- 如果前端选择“硅基流动”，应填写 `SILICONFLOW_API_KEY`，且前端大模型设置中选择硅基流动预设；默认官方 DeepSeek 使用 `DEEPSEEK_API_KEY`。

## 构建命令

```powershell
python tools/build_judge_release.py
```

常用参数：

```powershell
python tools/build_judge_release.py --skip-wheelhouse
python tools/build_judge_release.py --no-zip
```

输出位于 `04-输出版本存档/造价智算-评委运行版-YYYY-MM-DD-vX.X`，同时生成同名 `.zip`。

## 后续 Tauri MVP 可复用内容

- 后端同源托管能力：Tauri 可以继续打开 `http://127.0.0.1:8000/` 或内嵌本地前端。
- 前端生产 API 基址：生产构建默认使用同源 API，不依赖 Vite 开发端口。
- `.env.local.example` 与启动器环境加载逻辑：后续可迁移为 Tauri 设置页或本地配置文件。
- 绿色版包含 / 排除清单：可直接作为 Tauri 打包资源清单的第一版。
