# 造价智算 Windows 绿色版说明

## 定位

绿色版采用传统本地双服务启动：

- 后端：`http://127.0.0.1:8000`
- 前端：`http://127.0.0.1:5174`

它沿用开发版的前后端双服务逻辑，但随包携带 Python、Node 和前端依赖，目标 Windows 电脑不需要预装 Python、Node 或 npm。当前绿色版不使用桌面壳，也不把前端作为后端静态目录托管。

## 启动

双击：

```text
启动造价智算.bat
```

启动后会打开两个服务窗口，请保持窗口打开。停止时双击：

```text
停止造价智算.bat
```

## 进入绿色版的内容

- `backend/app/`：FastAPI 后端和业务逻辑。
- `frontend/`：React / Vite 前端源码、`package.json`、`node_modules`，以及 `public/file-viewer/vendor/docx/` 下的本地 DOCX Worker / JSZip；Word 报告预览断网运行，不访问 CDN 或在线 Office。
- `runtime/python/`：随包 Python。
- `runtime/python-libs/`：后端 Python 依赖。
- `runtime/node/`：随包 Node/npm。
- `runtime/wheels/`：Python 离线依赖包。
- `03-知识库-二维数据库制作/`：知识库、报告模板和可用样例。
- `03-【匹配规则】-勘察测绘知识库-匹配规则提炼/`：规则说明、原始资料 Markdown 和批注表。
- `05-经验池-预警数据/`：经验池模板、经验池数据和偏好配置。
- `config/project-default-settings.json`：表格预览、列映射、工作量抓取和问问智算窗口的集中项目默认配置；绿色版构建时必须随包复制，源文件缺失时打包脚本会生成一份带 `_说明` 的完整兜底配置，确保目标电脑首次打开也能生效。
- `business-skills/`：专业能力 Manifest、工作说明和验收说明。打包脚本会整体复制该目录，使绿色版与开发版使用同一份非秘密 Skill 清单；运行时不从 `SKILL.md` 解析价格规则。
- `.env.local.example`、启动 / 停止 / 状态 / 大模型配置脚本。
- 如果项目根目录存在 `.env.local`，打包时会随包复制，便于目标电脑直接使用问问智算。

## 不进入绿色版的内容

- `src-tauri/`、桌面壳源码、桌面壳构建产物和安装包。
- `frontend/dist/` 静态构建产物。
- `backend/tests/`、`.pytest_cache/`、`__pycache__/`。
- `04-输出版本存档/` 历史包。
- `src-tauri/target/` 等 Tauri 编译缓存。
- 项目代码存档仍不包含 `.env.local`；绿色版分发包是否带 Key 由打包时项目根目录是否存在 `.env.local` 决定。

## 构建命令

```powershell
python tools/build_green_release.py --date 2026-07-09 --skip-frontend-install
```

如果需要重新整理前端依赖，可去掉 `--skip-frontend-install`。

输出位于：

```text
04-输出版本存档/造价智算-绿色版-YYYY-MM-DD-vX.X
```

并生成同名 `.zip`。
