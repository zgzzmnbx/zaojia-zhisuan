# CHANGELOG

## v5.14.2 - 2026-07-22

- 完成 P0 简易多人复核：外部派发时可选择一名或多名复核人，编制人领取后通过经典卡片提交复核；每名指定复核人可通过或退回，全部通过后任务自动完成。
- 任一复核人退回后任务回到编制，编制人可重新提交并开启新轮次；逐人状态、复核轮次、结论时间和审计动作均持久化，重复点击保持幂等。
- 卡片操作统一按真实平台 Open ID 校验：只有指定编制人能领取和提交，只有当前任务复核人能给出结论；复核卡只投递到唯一授权“智算测试”群。
- 智能协同模拟器新增多人复核选择，台账同时展示编制人、复核人、轮次和复核卡状态；公开接口继续隐藏群 ID、用户 ID、文件 Key、路径和密钥。
- 结构化复核意见、成果冻结版本、网页复核工作台和精准单聊仍是后续增强，不在本次简易 P0 范围内。

## v5.14.1 - 2026-07-22

- 补齐智能协同 P0-A 领取闭环：WeAct / 飞书经典任务卡新增“领取任务”，长连接卡片回调以操作人真实 Open ID 校验指定编制人；非目标人拒绝，目标人重复点击幂等，领取状态和时间真实落库。
- 领取成功后原卡片更新为“已领取”并移除领取按钮；领取只改变任务流转状态，不批准、不修改专业数据、不触发计算或完成任务。
- 外部任务模拟器默认自动编写 `SIM-时间-随机码` 任务编号，项目名留空时自动补为“项目-XXXXXX”；后端接口同样提供缺省兜底。
- 派发台账新增“流程人员”列，展示角色、人员名称和领取状态；公开响应继续隐藏群 ID、用户 ID、文件 Key、运行路径和密钥。
- 验证完成：外部派发专项 `16 passed`，全量后端 `352 passed, 4 skipped`，前端工作台 `3 passed`、报告预览 `8 passed`、生产构建与 Python 编译均通过；内置浏览器已核对自动编号、缺省项目名、流程人员列、1000×720 页面无整体横向溢出且控制台零错误。
