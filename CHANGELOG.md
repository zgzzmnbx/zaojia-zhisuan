# CHANGELOG

## v5.8.3 - 2026-07-12

- 统一根目录、前端、后端、Tauri、Cargo、README、当前版本计划和健康检查的版本号至 `v5.8.3`。
- 发布此前已完成的飞书第二层机器人启停开关与知识库问答能力，保持问答与 Excel 文件任务分流、事件幂等，并确保不修改价格、系数或正式知识库。
- 本次仅做版本发布与存档，不改变核心匹配规则。

## v5.8.2 - 2026-07-12

- 第二层飞书机器人新增知识库问答入口：群聊 `@机器人 + @知识库：问题`，或机器人单聊 `@知识库：问题`，自动调用现有 `/api/knowledge/ask`，先回执查询中，再异步返回答案、来源文件和“不改变程序填价结果”的边界提示。
- 知识库问答与 Excel 文件任务分流，使用事件 ID 幂等记录；无证据、模型异常或错误消息均明确回传，不进入单任务文件队列，不修改 Excel、价格、系数或正式知识库。
- 更新智能协同模块 PRD、第二层机器人操作说明、页面能力提示和版本号至 `v5.8.2`。

## v5.8.1 - 2026-07-11

- 智能协同页新增第一层“启用通知”和第二层“启用接收”即时开关；第二层关闭会真正结束本机长连接，再开启会自动拉起机器人并轮询至“运行中”，无需重新双击启动脚本。
- 修复第二层状态标签和任务表格挤出卡片的问题；明确同一飞书应用的多长连接实例为随机集群分发而非广播，本地与服务器迁移时应只启用一个实例，避免任务和成果分裂。
- 新增“数字化项目助手”一级入口：宿主仅以 iframe 嵌入独立项目的 `5175` 前端，支持 `VITE_DIGITAL_PROJECT_ASSISTANT_URL` 覆盖地址，并提供加载、10 秒超时、重试和新窗口打开降级；不复制独立业务代码、不代理 `8010` API。
- 收紧 Word 报告页操作区：将报告状态、文件名、下载 / 刷新 / 风险清单操作和统计信息重排为紧凑响应式工具栏，保留真实 DOCX 预览与所有既有下载、刷新、风险清单能力。
- 增加 `frontend/.env.example`，明确独立数字化项目助手默认嵌入地址；项目默认设定文档同步明确集中配置优先级与例外说明要求。
- 版本同步至 `v5.8.1`；验证见本次发布记录。
