# CHANGELOG

## v5.8.10 - 2026-07-14

- 第二层机器人收到群聊 `@`消息或机器人单聊消息后，立即在原消息下方异步添加“了解”（`Get`）表情回应。
- 表情回应使用原消息 ID 和当前机器人的 REST API 域名，同时兼容普通飞书与企业 WeAct；重复投递在进程内去重。
- 表情回应失败只记录警告，不会阻断知识库问答、Excel 收件、顺序队列或专业处理链。

## v5.8.9 - 2026-07-14

- 第二层机器人运行控制台改为“刷新”旁按钮打开的弹窗，避免长期占用智能协同设置页空间。
- 消息日志新增发送人名称与 ID、群名与会话 ID、完整消息正文和附件名；通过飞书用户 / 群信息接口尽力解析名称并缓存，权限不足时回退显示 ID。长连接事件不提供来源 IP，控制台明确标注平台未提供。
- 业务日志仍隔离 App Secret、访问令牌、连接票据、access key、文件 Key 和完整 WebSocket 地址；机器人收件、顺序队列和专业处理链路保持不变。

## v5.8.8 - 2026-07-13

- 智能协同页新增第二层机器人运行控制台，实时汇总长连接、自动重连、消息接收、知识库问答和任务阶段日志，并显示当前机器人及进程状态。
- 新增只读日志接口与 2 MB 轮转结构化事件文件；原始 SDK 日志只提取安全状态，不回显访问令牌、连接票据、文件 Key、群 / 用户标识或完整连接地址。
