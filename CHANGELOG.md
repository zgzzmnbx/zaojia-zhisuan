# CHANGELOG

## v5.8.7 - 2026-07-13

- 第二层企业应用明确提供 `默认机器人（普通飞书）` 与 `Weact机器人（管网内网）` 两个选项，并新增按机器人选择 API / WebSocket 域名的能力；本机默认启用普通飞书，云端默认启用 Weact，用户可随时关闭或切换。
- 企业 WeAct 配置已完成真实访问令牌获取和 WebSocket 长连接验证；启动器立即记录子进程 PID，避免 SDK 初始化期间页面短暂误报“未运行”。

## v5.8.6 - 2026-07-13

- 第一层 Webhook 支持多配置选择，新增 `weact机器人（webhook）`，切换时保留通知开关、通知规则和发送历史，不向前端回显完整地址或签名密钥。
- 第一层 Webhook 与第二层企业应用机器人配置统一迁移到 `Codex-Temp/runtime/feishu-robot-settings.json`，继续兼容旧版分散配置格式，并保证两个配置分区互不覆盖。

## v5.8.5 - 2026-07-13

- 第二层飞书机器人支持多个企业应用配置选择；新增企业 Weact 配置（现界面名称为 `Weact机器人（管网内网）`），切换时先安全退出旧长连接，再启动选中的机器人，继续复用同一单任务顺序队列。
- 保持旧版单机器人凭证文件格式兼容；状态接口只返回机器人名称和 App ID 尾号，不回显 App Secret。
