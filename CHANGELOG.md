# CHANGELOG

## v5.12.0 - 2026-07-22

- 完成专业 Skill P0-B 运行时接口：新增任务级 `SkillRuntimeContext`，冻结输入规范、白名单处理器、结构化计价库、规则资产、经验池与阈值、知识源、报告模板、验证信息和能力开关；任务状态只保存项目内相对路径，公开接口不泄露真实资产路径。
- 现有勘察测量流程包装为代码白名单 `survey-measurement-v1`，继续复用同一 `FillEngine` 和三数字口径；批量匹配、辅助填价、经验池预警、任务内知识问答、风险报告、Word 重生成和下载均从任务快照恢复运行时上下文，不再重新读取当前默认 Skill。
- 报告生成支持显式 Skill 模板，知识检索支持按 Skill 知识源独立建索引，经验池预警按 Skill 风险配置读取；前端知识问答自动携带当前任务或所选 Skill 上下文。
- 新增测试专用第二 Skill 隔离夹具，验证不同计价库、技术规则、经验池、知识源和 Word 模板真实切换，并验证默认 Skill 改变后既有任务不漂移；该测试能力不进入生产清单，通用服务类仍待真实资料与数据。
- 验证完成：专业 Skill 专项 `19 passed, 1 skipped`；全量后端 `333 passed, 4 skipped`；前端 v5.12.0 生产构建通过。现有 100 行旧请求与显式 Skill 逐工作表、逐单元格一致。
- 内置浏览器运行态验收通过：v5.12.0 演示样例 101 / 101 匹配、0 行待复核、任务 Skill 已锁定；1433px 视口无横向溢出，控制台无错误。

## v5.11.0 - 2026-07-21

- 完成“对话式智能体工作台”P0：左侧新增“智算助手”一级入口，右侧智算 Dock 新增“进入智算助手”；两个入口进入同一宽屏会话工作台，只保留左侧导航，原专业工作区隐藏但不卸载。

- 会话内结构化动作复用现有真实业务链路：上传 / inspect、开始转换、批量匹配、经验池预警、风险清单、风险报告、Excel / Word 下载与专业页跳转；未修改三数字匹配、经验池规则、正式知识库或报告模板。
